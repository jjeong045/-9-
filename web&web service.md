## web&web service

**web service**란?? 			

> 요청(request)하고 응답,처리(response)하는 시스템
>
> 클라이언트(client) ----> 서버(server)
>
> 오가는 문서 ==> HTML

**web** 란??

WWW = **w**orld **w**ide **w**eb 세상 넓은 거미줄 ==> 인터넷

**web brouser** 란??

* 클라이언트(client)와 서버(server) 사이에 통로 역할
* HTML 파일을 주고 받는다.
